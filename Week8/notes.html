Chapter 8 - Transforms and Transitions

Transforms

Transform property lets you translate, rotate, scale, skew any element.

Translation allows you to move elements left, right, up, or down.


            transform: translate(45px, -45px);

Transforms don't work on inline elements. Add display: inline-block;

Scale(x,y) function scales an element by the defined factors horizontally then vertically. One value provided will be used for X and Y values.

The rotate() function rotates an element around the oint of origin by a specified angle value. The default point of origin is the element's center.

Skew(x,y) function specifies a skew along the x and y axes.

The transform-origin property changes where the point of origin is - works for ellipses not circles

Order of transform functions matter - if you rotate before translating, your translate direction will be on the rotated axis.

If you want to rotate in IE go to IE Transforms Translator

Transitions

We can animate any of the transforms we've seen so far.

Steps to create a simple transition using only CSS:

  1. Declare the original state of the element in the default style declaration.
  2. Declare the final state of your transitioned element; for example, a :hover state.
  3. Include the transition functions in your default style declaration using the transition properties, including: transition-property, transition-duration, transition-timing-function, and transition-delay.

It's important that the transition is declared in the default state. -webkit- vendor prefix is needed for older mobile devices.

Transition-property defines the CSS properties of the element that should be transitioned, will "all" for all properties being the default

For example, we can transition color and width

List of animatable properties can be found https://developer.mozilla.org/en-US/docs/Web/CSS/CSS_animated_properties


            transition-property: transform;

Transition-duration property sets how long the transition will take in s or ms

Transition-timing-function lets you control the pace of the transition.

  • Ease
  • Linear
  • Ease-in
  • Ease-out
  • Ease-in-out
  • Cubic-bezier(0, 0, 1, 1) (this one is the same as linear, but can be customized)

Transition-delay introduces a delay before the transition begins. Default is 0.

Transition shorthand property order:

  • transition-property
  • transition-duration
  • transition-timing-function
  • transition-delay

            transition: transform 0.2s ease-out 50ms;

With multiple transitions you can separate values with commas


            transition-property: transform, color;
            transition-duration: 0.2s;
            transition-timing-function: ease-out, linear;
            transition-delay: 50ms;

Or with shorthand:


            transition: transform 0.2s ease-out 50ms, color 0.1s linear 50ms;

Animations

You control animations via keyframes


            @-webkit-keyframes myAnimation {
                /* Animation keyframes here */
            }
            @keyframes myAnimation {
                /* Animation keyframes here */
            }

Examples:


            @keyframes moveRight {
                from {
                    transform: translateX(-50%);
                }
                to {
                    transform: translateX(50%);
                }
            }
            @keyframes appearDisappear {
                0, 100% {
                    opacity: 0;
                }
                20%, 80% {
                    opacity: 1;
                }
            }

Animation properties include animation-name and animation-duration

Animation-timing-function options are the same as for transition-timing-function

Can also use steps(number, direction)

animation-iteration-count lets you define how many times the animation will play. Default is 1, can be up to infinite

animation-direction default is normal, can do reverse, or alternate

animation-delay how many seconds to wait before browser begins animation

animation-fill-mode defines what happens before the first animation iteration begins. none, forwards, backwards, or both. Default is none

Animation shorthand:


            .verbose {
                animation-name: appearDisappear; 
                animation-duration: 300ms; 
                animation-timing-function: ease-in;
                animation-iteration-count: 1;
                animation-direction: alternate;
                animation-delay: 5s;
                animation-fill-mode: backwards;
                animation-play-state: running;
            }
            /* shorthand */
            .concise {
                animation: 300ms ease-in alternate 5s backwards appearDisappear;
            }

Drag and drop

Two major kinds of implementation with drag and drop: drag files from computer into a web page in combination with the File API, or dragging elements into other elements on the same page

A guide to using File API drag and drop

Several steps to adding drag and drop to your page:

  1. Set the draggable attribute on any HTML elements you'd like to be draggable
  2. Add an event listener for the dragStart event on any draggable HTML elements
  3. Add an event listener for the dragover and drop events on any elemnts that you want to have accept dropped items.
