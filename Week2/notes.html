Chapter 2

  • Using Methods with Numbers:
  • Use two dots
    • 5..toExponential();
  • Put a space before the dot:
    • 5 .toExponential();
  • Always write integers as a decimal:
    • 5.0.toExponential();
  • Place the integer in parentheses
    • (5).toExponential();
  • Assign the number to a constant:
    • const number = 5;
    • Number.toExponential();

Null behaves like the number 0, undefined is not a number

Chapter 3

Destructuring array:

  • Const [x,y] = [1,2]; // x = 1, y = 2
  • [x,y] = [y,x] // x = 2, y = 1

Array Methods:

  • .pop() returns the last item of the array and deletes it
  • .shift() returns and removes the first item of the array
  • .push() appends a new value to the end of the array
  • .unshift() adds a new item to the beginning of the array
  • .concat() merges an array with one or more arrays
  • You can also spread arrays. Example
    • Avengers = […avengers, …['Hulk', 'Hawkeye', 'Black Widow'] ]; // Adds to the avengers array
  • .join() turns array into a string separated by commas, or you can add your own .join(' & ');
  • .slice() creates a subarray
  • .splice() removes items from the array and adds something new in its place
  • .reverse() self explanatory
  • .sort() Alphabetically (numbers included-by first digit)

Set includes numbers that don't duplicate, or if a string is used it contains all letters in the string, not repeating.
