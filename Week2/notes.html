Chapter 2

  • Using Methods with Numbers:
  • Use two dots
    • 5..toExponential();
  • Put a space before the dot:
    • 5 .toExponential();
  • Always write integers as a decimal:
    • 5.0.toExponential();
  • Place the integer in parentheses
    • (5).toExponential();
  • Assign the number to a constant:
    • const number = 5;
    • Number.toExponential();

Null behaves like the number 0, undefined is not a number

Chapter 3

Destructuring array:

  • Const [x,y] = [1,2]; // x = 1, y = 2
  • [x,y] = [y,x] // x = 2, y = 1

Array Methods:

  • .pop() returns the last item of the array and deletes it
  • .shift() returns and removes the first item of the array
  • .push() appends a new value to the end of the array
  • .unshift() adds a new item to the beginning of the array
  • .concat() merges an array with one or more arrays
  • You can also spread arrays. Example
    • Avengers = […avengers, …['Hulk', 'Hawkeye', 'Black Widow'] ]; // Adds to the avengers array
  • .join() turns array into a string separated by commas, or you can add your own .join(' & ');
  • .slice() creates a subarray
  • .splice() removes items from the array and adds something new in its place
  • .reverse() self explanatory
  • .sort() Alphabetically (numbers included-by first digit)

Set includes numbers that don't duplicate, or if a string is used it contains all letters in the string, not repeating.

Convert set to array -> spread again

Const shoppingArray = […shoppingSet];

Weak Sets can be used to garbage collect references to a 'dead object' that had its original reference removed

Maps can keep a list of key and value pairs, like a hash table

Chapter 4

Ways to declare functions:

            Function hello(){
                Console.log('Hello World!');
            }
        
            Const goodbye = function() {
            Console.log('Goodbye World!');
        }; // This is an anonymous function because it doesn't have a name, it is only assigned to the variable 'goodbye'
        
            Const goodbye = function bye(){
                Console.log('Goodbye World!');
            }; //These examples end with a semicolon

Rest operator creates an array of arguments in the function. Example:

            Function rest(…args){
                Return args;
            } // The args parameter is an actual array and has access to the same methods

You can include default parameters for an unused variable in a function:

            Function hello(name='World') {
                Console.log(`Hello ${name}!`);
            }

If you call hello() it defaults to Hello World! Or if you call hello('Universe'); You get 'Hello Universe!'

Put default parameters after all the other parameters.

Arrow functions: anonymous, succinct

            Const square = x => x*x; // Takes value x and returns x*x - no need for return statement
                Const add = (x,y) => x + y; // Requires parenthesis for more than one parameter
                Const hello = () => alert('Hello World!'); // No parameters needs empty parenthesis

Longer functions require curly braces:

forEach() method loops through an array and invokes a callback function using each value as an argument.

            Colors.forEach( (color,index) =>
                Console.log(`Color at position ${index} is ${color}`) );

Map() function is similar to forEach(), takes a callback function as a parameter that is invoked on each item in the array.
