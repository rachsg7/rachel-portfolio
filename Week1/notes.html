Notes

"Whatever you are doing - do mobile first." - Eric Schmidt

Responsive retrofitting - take desktop and then work on mobile fitting

Mobile first is time consuming and costly for a full rebuild

Provides a better experience across devices Future friendly

Things grow well from S->L

Decrease steps if possible. 0 Steps is better than 5 steps

Reachability matters - where thumbs go Speed matters - optimize or have drops in traffic and revenue

Reduce image usage and use SVG/Webfonts Optimize

CSS and JS files GZIP files Menu on bottom - easier to use Hamburger menu - not intuitive for everyone

Forms: float labels, bigger buttons, bigger checkboxes, break into sections, enable autocomplete, space numbers in credit card info, give users the best keyboard for the form, don't list phone numbers as text

No pop up ads!!! Make ads small and simple, not 40% of screen real estate. Window of content ad, simple effective, not blocking.
